# IBS-LK Business Manager — v1.8.0 Product Control Center Handoff

**Release:** v1.8.0  
**Prior:** v1.7.1 Live Product and Order Sync Connection Test  
**Date:** 2026-06-07

---

## What changed

### Product list
- Server-side filters via GET: search, product ID, name, OC model, supplier model, health chips
- **20 rows per page** with Previous/Next pagination
- Compact table with **thumbnail** column
- Standard health badges: Ready, Needs Work, Low Stock, Missing Cost, Missing Supplier Model, Sync Required

### Product Control Center modal
- Sync information panel: last synced, source product ID, source sync status
- Completeness badges in header (Ready / Needs Work + detail badges)
- Read-only OC fields: ERP product ID, name, model, stock, image, source product ID
- OC product status shows **Not stored in ERP** (no migration in this release)
- Variable products: collapsible **Options (N)** accordion with option image, name, value, OC model/stock
- Supplier fields editable: model, cost, stock, low warning, note (0012 gate), active/inactive

### Unchanged safety
- Sync-only catalog — manual product/variant create blocked
- No product delete
- No OpenCart write
- Re-import updates platform fields only (`updatePlatformSyncFields`)

## Operator QA

1. Import demo products from `/sync-preview`
2. `/product-control` — verify filters, pagination, thumbnails, badges
3. Open product — verify modal fields and variant accordion with images
4. Save supplier fields — re-import from sync preview — supplier fields preserved
5. Run `powershell -NoProfile -ExecutionPolicy Bypass -File tools\check-local.ps1`

## Not in this release

- No new migrations
- No order/sync/dispatch/payable/invoice changes
