## What v1.8.6 delivers

### Sidebar structure

- **Dashboard** — standalone
- **Fulfillment** — Orders, Dispatch Reports, Return Receive, Manual Orders, Product Control, Pre-acquisition (disabled placeholder)
- **Finance** — Payables, Settlements, Opening Balance
- **Reports** (collapsed) — Sales/Order, Supplier, Return, Product Cost/Stock report links via `?report=`
- **Settings** (collapsed) — Suppliers, Business Sources, Status Mapping, Sync/API Settings, Users, Roles
- **Tools** (collapsed) — Quick Invoice, Calculator (modal)
- **System** (collapsed) — Activity Log, Health, Database Safety
- **Developer** (collapsed, non-prod) — migration/dev tools

### Removed from daily menu

Sync Preview, Status Mapping (from primary), Invoice Printing, Offline Invoices, Version — routes still work by direct URL.

### Product sync embedded in Product Control

Collapsible **Product Sync / Import** panel on `/product-control` with preview, import, and reset. POST endpoints unchanged on `/sync-preview/*`.

### Order sync

Stays on `/sync-preview` (hidden from menu) with relocation banner. Future: embed in Orders page.
