# IBS-LK Business Manager — v1.0.0 "Production Launch" Handoff

**Release tag:** `1.0.0`  
**Prior release:** `0.8.0` Release Candidate Hardening (`d927d6b`)  
**Date:** 2026-06-07

This is the first **production-ready** version of IBS-LK Business Manager for Lokkisona / Sonamoni supplier operations (Iqbal & Brothers). Code is complete for daily supplier fulfillment; **live domain cutover** requires owner steps on the server (credentials, DB migrations, opening balance, controlled OpenCart sync).

---

## What v1.0.0 delivers

End-to-end supplier ERP (channel-neutral):

1. Manual + synced orders → order workflow
2. Daily dispatch report → locked cost snapshot
3. Supplier payables → owner-approved ledger
4. Return receive → return batch → deduction draft
5. Settlements → period close
6. Reports (print/CSV), invoice printing, supplier tools
7. Controlled OpenCart Test Sync (owner-approved import, max 50)
