# IBS-LK Business Manager — v1.1.0 "UI/UX Revamp" Handoff

**Release tag:** `1.1.0`  
**Prior release:** `1.0.0` Production Launch (`e25600c`)  
**Date:** 2026-06-08

Visual refresh only — NexaERP-inspired clarity for daily operations. **No business-logic, schema, route, or migration changes.**

---

## What v1.1.0 delivers

### Tiered admin navigation

| Tier | Visibility | Groups / items |
|------|------------|----------------|
| **Primary** | Always visible (~12 links) | Fulfillment (Dashboard, Order Workflow, Manual Orders, Dispatch Reports, Return Receive), Finance (Supplier Payables, Settlements, Reports), Catalog (Product Control, Status Mapping, Sync Preview, Invoice Printing) |
| **Setup & Admin** | Collapsed `<details>` | Users, Suppliers, Business Sources, Supplier Opening Balances, Supplier Tools |
| **System** | Collapsed `<details>` | Activity Log, Health Check, Version, Role & Permissions |
| **Developer** | Collapsed; **hidden when `env === production`** | Database Safety, Dev DB Activation, Migration tools, Build Queue |

**Supplier role:** unchanged — 4 primary links (Dashboard, Orders, Returns, Supplier Tools). No developer tiers.

### Owner dashboard command center

- Welcome strip: `Welcome back, {user} — {date}`
- 6 KPI cards (3×2 grid): Active Orders, Shipped Awaiting Dispatch, Net Payable, Pending Returns, Payable Drafts, Dispatch Snapshots
- Widget row: Order Status Breakdown + Needs Attention
- Recent Activity (from ActivityLog, max 5)
- Removed: Quick Control link dump, System card, Dev DB link, Planning Foundation collapsible, sync_status dev string

### Environment gating

- DEV/TEST banner: only when `config('app.env')` is `local` or `staging`
- Developer nav tier: hidden on `production`

### Dark + light themes

All new components use CSS variables (`--color-*`, `--kpi-accent-*`). Toggle unchanged (`ibs-theme` in `public/assets/js/app.js`).
