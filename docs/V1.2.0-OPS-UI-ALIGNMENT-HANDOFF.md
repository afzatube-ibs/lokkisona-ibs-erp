## What v1.2.0 delivers

### Shared components

- [`resources/views/partials/ops-safety-strip.php`](resources/views/partials/ops-safety-strip.php) — token-based safety banner (replaces inline styles)
- `.ops-safety-strip`, `.ops-page-subtitle` CSS classes in [`public/assets/css/app.css`](public/assets/css/app.css)
- `.kpi-grid-4` modifier for 4-column KPI rows (settlements)
- Workflow stage cards: accent top border on active stage (matches dashboard KPI style)

### Operations pages updated

| Page | Changes |
|------|---------|
| `/supplier-payables` | KPI grid (3 cards), compact header, safety partial |
| `/settlements` | KPI grid (4 cards), compact header |
| `/sync-preview` | Preview counts in KPI grid, compact header |
| `/dispatch-reports` | Safety partial, compact header |
| `/return-receive` | Safety partial, compact header |
| `/order-workflow` | Compact header, stage card accent borders |

### Setup-tier pages updated

Planning and read-inventory reference content collapsed by default:

- `/users`
- `/roles-permissions`
- `/suppliers` (write forms remain visible)
- `/business-sources` (write forms remain visible)
- `/supplier-opening-balances` (write forms remain visible)
