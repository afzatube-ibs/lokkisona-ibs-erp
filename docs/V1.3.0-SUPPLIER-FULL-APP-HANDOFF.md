## What changed

### SupplierContext

- New [`app/SupplierContext.php`](app/SupplierContext.php)

### Permissions & navigation

Supplier role now includes:

- `dispatch_reports.view` (read-only)
- `supplier_payables.view` + `supplier_payables.manage` (manual ledger drafts)
- `settlements.view` (read-only)
- `product_control.view` + `product_control.manage`
- `manual_orders.view` + `manual_orders.manage`

Blocked: developer tier, sync, users, suppliers setup, opening balances, **manual order creation** (online orders are owner-only).

Primary nav groups: Fulfillment, Finance, Catalog (8+ links). **Offline Invoices** is a Finance nav item â€” not buried in Tools.
