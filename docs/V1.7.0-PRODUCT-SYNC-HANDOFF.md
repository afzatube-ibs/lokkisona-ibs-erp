**Release tag:** `1.7.0`  
**Prior release:** `1.6.2` Modern SaaS UI Phase 3  
**Date:** 2026-06-07

## What changed

### Product Control list/edit completion

- Catalog table columns: Product Name, OC Source ID, Category, Last Synced (plus existing model/stock/health columns).
- Modal workspace: ERP status, owner supplier assignment, supplier note (column-gated), cost/stock history on save.
- Variable products without synced options show **No option synced** badge and modal notice.

### Lokkisona product sync (read-only)

- Warehouse pull upserts **parent products + option/variant lines** when OpenCart returns `options[]`.
- Max **50 parent products** per pull (unchanged).
- Re-pull updates platform fields only — supplier model, cost, stock, warning, note preserved.
- Demo products 501 (simple), 503 (variable with options), 504 (variable, no options) in `config/opencart.php`.
- `default_supplier_id` on new pull rows (config key).

### Sync Preview

- **Product Sync Status** card: mode, route, pull availability, rules.

### Migration 0012 (draft, manual only)

`database/migrations/0012_supplier_product_note.sql`:

- `supplier_note` on `ibs_products` and `ibs_product_variants`
- `sync_options_state` on `ibs_products` (`has_options` / `missing_options` / `simple`)

**Not auto-applied.** Product list/edit works without 0012; supplier note fields hidden with gate message.

## Operator guide — demo/local

1. Keep `demo_mode=true` in `config/opencart.php`.
2. Open `/sync-preview` → verify **Product Sync Status** shows demo mode.
3. Click **Pull warehouse products**.
4. Open `/product-control` — expect demo products 501 and 503 (504 after 0012 for "No option synced" state).
5. Open a product row → edit supplier fields → **Save All Changes**.
6. Re-pull → OC name/stock update; supplier fields unchanged.

## Operator guide — live staging

1. Apply migrations 0003, 0004, 0011 (+ 0012 if owner approved).
2. Set `config/opencart.php`: `enabled=true`, `demo_mode=false`, API key, `product_api_route`.
3. OpenCart products: **From Warehouse = Yes**.
4. Pull warehouse products → edit supplier fields on Product Control.
5. Order Test Sync / import unchanged (smoke only).
