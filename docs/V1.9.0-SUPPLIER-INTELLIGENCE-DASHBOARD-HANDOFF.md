# IBS-LK Business Manager — v1.9.0 Supplier Intelligence Dashboard Revamp

**Release:** v1.9.0  
**Prior:** v1.8.7.4 Product Control Center Finalization  
**Date:** 2026-06-08

---

## What changed

### Dashboard (`/dashboard`)
- **Header:** IQBAL & BROTHERS (IBS) — Supplier Intelligence & Fulfillment Command Center; supplier/source meta; compact **IB** avatar badge (no logo image block)
- **Insight strip:** week-over-week and operational highlights with link to Reports
- **8 KPI cards:** orders MTD, dispatch reports, delivered, payable, vendor stock value, low stock, return rate, IBS performance score
- **Performance score:** weighted composite (40% dispatch SLA, 25% stock readiness, 15% return quality, 10% data, 10% cost)
- **Dispatch SLA:** **Order Received → Created Report** (12h target — not courier delivery)
- **Payable command center:** ledger lines (BDT) + status cards
- **Trend chart:** orders, dispatch, payable, forecast series
- **Top products / categories, catalog health, return intelligence, priority actions**

### Backend
- New read service: `SupplierIntelligenceDashboardService`
- `DashboardController` scopes owner view to `config('auth.supplier_id', 1)` (IBS), same as Product Control pattern
- Legacy `dashboard/overview.php` retained but unused

### Unchanged safety
- Read-only dashboard — no Order Workflow mutations
- No OpenCart writes, stock deduction, or page-load schema changes
- No new migrations
