## What changed

### Product list
- **Product column:** thumbnail + **OC model (bold)** + OC ID subline — full product name hidden but still searchable via `q` / filters
- **Removed** separate Model column (no duplicate Owner Model No)
- **Compact table:** fixed column widths, tight padding, ellipsis; Average Cost / Owner Stock hidden on narrow viewports
- **20 rows per page** unchanged

### Product Control Center modal
- **Hero split:** compact OC read-only facts (left) + **Iqbal & Brothers (IBS)** supplier card (right)
- **Supplier view mode:** read-only chips for Supplier Product Name (synced), Main Vendor Model, Low Stock (+ simple cost/stock when set)
- **Edit:** Edit button or double-click supplier card → inputs → **Save All Changes**
- **Variant section:** flat lines table — Line, Image, Model, Vendor Model, OC Price, Average Cost, Owner Stock, Vendor Stock, Warning, Health; row edits for vendor model, cost, vendor stock

### Unchanged safety
- Sync-only catalog — no manual product/variant create or delete
- No OpenCart write
- Re-import / Refresh Products updates platform fields only (`updatePlatformSyncFields`)
